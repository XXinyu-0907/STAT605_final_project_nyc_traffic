  • We found a 24GB data set on Kaggle. Here is the URL: https://www.kaggle.com/datasets/aadimator/nyc-realtime-traffic-speed-data . This data set contains real-time traffic information from locations where NYCDOT picks up within the five boroughs, mostly on major arterials and highways.
  • According to the topic we want to study initially: Explore which streets in New York have a higher rate of vehicles over speed. In this way, it is hoped that more speeding interventions can be done on these streets to reduce the incidence of traffic accidents.
  • In order to control the consistency in the choice of vehicle exceeding speed, we will then control the variable that only select the NYC_DOT_LIC level in the owner variable, and ignore the remaining 22% measurement on bridges or tunnels level from this variable in the future.
  • Besides, we divide several small data sets according to different levels of the LINK_ID variable, which represents different streets in New York. We split the data sets through R. In next step, for these 160 small data sets after segmentation, they will be then processed in parallel with the shell command on the CHTC platform to compute the rate of vehicles exceeding rate in each street.
  • It can be seen from the summary that the filtered data set has a total of 64914523 rows and 13 columns.
  • We will use the id speed link id borough link name these five variables. Speed stands for average speed a vehicle traveled between end points on the link in the most recent interval. The correspondence between id and link id is the basis for us to divide several small data sets. According to the correspondence between the link id and the link name, we can know the names of the streets with high rate of exceeding speed in New York. Borough represents the name of the New York district where the speed test street is located, including Brooklyn (11123764 data), Bronx (7676967), Manhattan (12547339), Queens (18944931) and Staten Island (14621522). In the future, the regional districts where high-speed streets located can be further explored for certain defenses.
